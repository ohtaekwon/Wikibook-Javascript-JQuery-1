## :pencil:_Lesson 01. 초보자를 위한 함수_

### _01. 함수란?_

- 함수는 `동전 교환기`와 같다.
  - ex) 1,000 투입 → 100원 10개 교환
- 함수는 특정 기능을 하는 `구문(알고리즘, 로직)`을 묶어 재사용하는 문법이다.

### _02. 함수는 이럴 때 사용한다._

> **1) 함수사용 전**

- 3단

```javascript
var dan = 3;

document.write(dan+ " X 2 = " +(dan*2), "<br>");
document.write(dan+ " X 3 = " +(dan*3), "<br>");

```

- 만약 4단, 5단, 6단, 7단, 8단, 9단 일일히 다 코드를 복사하고 붙여넣어주어야 하는 단점이 발생
- 일이 비효율적으로 된다. 이럴때 함수로 지정해주면 편리하다.

<br>

>**2)  함수사용 후**

```javascript

    // 이곳에 특정 기능을 하는 구문이 위치하게 된다.

    document.write(dan+ " X 2 = " +(dan*2), "<br>");

    document.write(dan+ " X 5 = " +(dan*5), "<br>");
    document.write(dan+ " X 6 = " +(dan*6), "<br>");
    document.write(dan+ " X 7 = " +(dan*7), "<br>");
    document.write(dan+ " X 8 = " +(dan*8), "<br>");

print99DAN(3);
print99DAN(4);
```

- 함수는 특정 기능을 하는 알고리즘을 재사용하기 위해 사용하는 포장기술로서,
- 중복되는 구구단 출력 알고리즘을 함수에 담아 적용시키면, 더욱 편리해진다.

### _04. 함수 호출_

- 함수만 만든다고 함수가 자동으로 동작하지는 않는다.
- 함수를 동작시키기 위해서는 `함수호출` 이라는 것을 해야한다.
  - 방법
    - 함수 이름 다음에 `괄호`를 열고 닫고 해주면 된다.
    - ex) 함수이름(); / alert(), document.write(), console.log()

<br>

### _05. 매개 변수_

- 함수가 호출되어 실행되기 시작하면 함수 내부는 함수 외부에서 접근할 수 없게 된다.![](https://user-images.githubusercontent.com/75871005/125598028-139370b9-370e-42a5-a286-5ede1d46e73d.png)

- 이때 함수 외부에서 함수 내부로 값을 전달하는 방법은
  - **매개변수**를 이용하는 것.
    - 외부 데이터를 함수 내부로 전달하는 매개체 역할하는 변수![](https://user-images.githubusercontent.com/75871005/125627493-a37f3849-67d8-4aed-b8bf-3647826cd769.png)

- 함수를 호출하면 `데이터`가 `매개변수1`로 넘어간다.
- 즉, `매개변수1 = 데이터` 와 같게 되는 것이다.

<br>

- 매개변수의 위치는 다음과 같이 괄호 안에 위치한다.![](https://user-images.githubusercontent.com/75871005/125628085-77ce9f2d-6725-4378-9ac9-bd35ff15ea92.png)
- 매개변수는 함수에 따라 없을 수도 있고 1개 이상 있을 수도 있다.

<br>

### _06. 리턴값_

- 리턴값은 매개변수와 반대의 개념이라고 보면 된다.![](https://user-images.githubusercontent.com/75871005/125629020-7b963dd3-f6d9-4e3f-b886-edb72ac96e72.png)

- `리턴값`은 **함수 내부에서 함수 외부**로 데이터를 보내기 위해 사용하는 통로이다.

```javascript

```

## :pencil:_Lesson 02. 초보자를 위한 클래스_

### _01. 클래스란?_

- **함수는 특정 기능을 하는 구문(알고리즘, 로직)을 묶을 때 사용하는 문법**이지만,
- **클래스**는 연관 있는 변수와 함수를 하나로 묶을 때 사용하는 문법이다.

![](https://user-images.githubusercontent.com/75871005/125636604-10d0b354-2781-4db6-95d2-c20011e231d8.png)

<br>

### _02. 클래스는 이럴 때 사용한다._

- 먼저 구현해야 할 UI 요소
  - 2단 메뉴
  - 이미지 슬라이더
  - 탭 패널
  - 아코디언 메뉴
  - 롤링배너

<br>

- 함수 기반 코딩의 단점
  - 코드가 구분없이 작성돼 있기 때문에 문제 발생 시 모래밭에서 읽어버린 진주를 찾는 것처럼 문제 발생 코드를 쉽게 찾을 수 없다.
  - 큰 프로젝트는 여러 사람이 협업을 이뤄 진해아헥 되는데 이떄 개발자끼리 똑같은 이름을 가진 변수와 함수를 만들어 충돌하는 문제점이 발생한다.
- 현실 세계에서 시, 도, 군 개념을 두어 관리를 하듯 프로그래밍에서 **연관 있는 요소들을 포장**해 관리하는 문법을 **클래스**라 한다.

<br>

### _04. 인스턴스와 객체_

- 클래스를 사용하기 위해서는 **인스턴스(객체)**를 생성해야 한다.
- 이 때, `new`라는 키워드를 사용한다.

```javascript
var tabMenu = new TabMenu();
```

<br>

### _05. 메서드와 프로퍼티_

- 일반 변수, 함수를 클래스 내부에 들어 있는 변수
- 함수와 구분하기 위해 클래스 내부에 있는 변수를 프로퍼티라 한다.![](https://user-images.githubusercontent.com/75871005/125663523-3005d4dc-b3b2-4476-a1f7-9202d38edc6e.png)

<br>

### _06. 클래스 내부에 들어 있는 함수(메서드)를 호출하는 방법_

- 함수와 마찬가지로 클래스 역시 클래스 외부에서 클래스 `내부의 변수(프로퍼티)`와 `함수(메서드)`를 마음대로 접근할 수 없다.
- 이 때, 접근하는 방법은 바로 `접근연산자(.)`를 활용하는 것이다.
- 즉, `접근연산자(.)`는 클래스 내부에 들어 있는 변수와 함수에 접근할 떄 사용하는 연산자이다.
- **클래스 내부에 들어있는 변수와 함수에 접근하기 위해서는** 반드시 `접근연산자(.)`를 사용해야 한다.

![](https://user-images.githubusercontent.com/75871005/125664966-830fecc0-3a7e-49ba-853c-2772773ab227.png)

***EX)***

```javascript
function Calculator(){
    this.add = function(a,b){
        alert("두 수의 합은 " +(a+b)+ "입니다");
    }
    this.sub=function(a,b){
        alert("두 수의 차는 " +(a-b)+ "입니다.");
    }
    this.mul=function(a,b){
        alert("두 수의 곱은 " +(a*b)+ "입니다.");
    }
    this.div=function(a,b){
        alert("두 수의 나눈 값은 " +(a/b)+ "입니다.");
    }
}
```

- 클래스 내부를 살펴보면 총 4 개의 사칙 연산 함수가 포장돼 있다.
- 이때 4개의 함수 중 두 수의 합을 알려주는 기능인 `add()`함수를 사용하고 싶다면 다음의 순서를 거쳐야 한다.

<br>

#### :ballot_box_with_check:_단계 01)_ 인스턴스 만들기

- 계산기에 들어 있는 함수를 사용하려면 `new`라는 키워드를 이요앻 계산기 클래스의 인스턴스를 만들어야 한다.

```javascript
var cal1 = new Calculator();
```

<br>

#### :ballot_box_with_check:_단계 02)_ 접근 연산자를 이용한 함수(메서드)호출

- 다음으로 접근 연산자를 이용해 원하는 기능인 `add()` 함수에 접근해서 호출해 준다.

```javascript
var cal1 = new Calculator();
cal1.add(10,20);
```

- 클래스 내부에 들어 있는 기능을 이용하기 위해서는 ***"인스턴스 생성 후 접근 연산자를 이용해 사용한다."***

<br>

### _07. 해석하기_

#### :one: document.write(); 해석

1. `.` 앞에 영단어가 있고 `.`뒤에는 함수 호출을 의미하는 괄호가 있다.
2. 따라서 `.`은 접근 연산자이며,
3. **document는 객체**이고, 이 안에 수 많은 기능(함수)가 있으며,
4. 그 중 write()라는 함수를 호출 한 것.

<br>

#### :two: console.log(); 해석

- 정리해보면, **console이라는 객체의 기능 중 log()라는 함수를 사용한 것**

```
영단어조합 + 점(.) + 함수이름();
```
